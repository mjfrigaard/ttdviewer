# Report Download Server Module with Enhanced Quarto Error Handling

Report Download Server Module with Enhanced Quarto Error Handling

## Usage

``` r
mod_report_download_server(id, format, data, dataset_title, selected_plot_type)
```

## Arguments

- id:

  Module ID

- format:

  A reactive expression returning the report format

- data:

  A reactive expression returning the dataset list

- dataset_title:

  A reactive expression returning the dataset title

- selected_plot_type:

  A reactive expression returning the selected plot type
