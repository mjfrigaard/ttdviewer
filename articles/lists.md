### ctr()

[`ctr()`](https://mjfrigaard.github.io/ttdviewer/reference/ctr.md) is a
custom folder tree function made with
[`lobstr::tree()`](https://lobstr.r-lib.org/reference/tree.html):

``` r
ctr <- function(x = as.list(), depth = 10L, length = 50L) {
  lobstr::tree(x,
    max_depth = depth,
    max_length = length,
    show_environments = FALSE,
    show_attributes = FALSE,
    hide_scalar_types = TRUE,
    class_printer = crayon::blue,
    remove_newlines = TRUE,
    val_printer = crayon::green
  )
}
```

``` r
ctr(x = ttd2)
```
